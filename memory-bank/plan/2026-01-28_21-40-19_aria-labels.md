## Goal

**Add ARIA accessibility attributes to the Jokes By JC Chrome extension** so screen reader users can navigate and use all interactive elements effectively.

**Non-goals:**
- Full WCAG 2.1 AA compliance audit
- Keyboard navigation improvements (separate ticket)
- Color contrast fixes

---

## Scope & Assumptions

### In Scope
- Add `aria-label` attributes to all buttons (static and dynamically created)
- Add `aria-live="polite"` region for joke text updates
- Verify avatar image has appropriate `alt` text

### Out of Scope
- Focus management/keyboard trapping
- High contrast mode
- Skip links

### Assumptions
- Extension already has functioning buttons with visible text
- No existing ARIA attributes to conflict with
- Testing will use manual screen reader verification
